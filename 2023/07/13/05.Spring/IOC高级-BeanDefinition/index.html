IOCй«ҳзә§-BeanдёҺBeanDefinition

BeanDefinitionжҰӮиҝ°

BeanDefinition д№ҹжҳҜдёҖз§Қй…ҚзҪ®е…ғдҝЎжҒҜпјҢе®ғжҸҸиҝ°дәҶ Bean зҡ„е®ҡд№үдҝЎжҒҜгҖӮ

bean зҡ„е®ҡд№үдҝЎжҒҜеҸҜд»ҘеҢ…еҗ«и®ёеӨҡй…ҚзҪ®дҝЎжҒҜпјҢеҢ…жӢ¬жһ„йҖ еҮҪж•°еҸӮж•°пјҢеұһжҖ§еҖје’Ңзү№е®ҡдәҺе®№еҷЁзҡ„дҝЎжҒҜпјҢдҫӢеҰӮеҲқе§ӢеҢ–ж–№жі•пјҢйқҷжҖҒе·ҘеҺӮж–№жі•еҗҚз§°зӯүгҖӮеӯҗ bean е®ҡд№үеҸҜд»Ҙд»ҺзҲ¶ bean е®ҡд№үз»§жүҝй…ҚзҪ®ж•°жҚ®гҖӮеӯҗ bean зҡ„е®ҡд№үдҝЎжҒҜеҸҜд»ҘиҰҶзӣ–жҹҗдәӣеҖјпјҢжҲ–иҖ…еҸҜд»Ҙж №жҚ®йңҖиҰҒж·»еҠ е…¶д»–еҖјгҖӮдҪҝз”ЁзҲ¶ bean е’Ңеӯҗ bean зҡ„е®ҡд№үеҸҜд»ҘиҠӮзңҒеҫҲеӨҡиҫ“е…Ҙ

bean зҡ„е®ҡд№үдҝЎжҒҜд№ҹжҳҜжңүеұӮж¬ЎжҖ§зҡ„пјҲиҒ”жғі BeanFactory зҡ„еұӮж¬ЎжҖ§пјүпјҢbean зҡ„е®ҡд№үдҝЎжҒҜеҸҜд»Ҙз»§жүҝиҮӘжҹҗдёӘе·Із»Ҹжңүзҡ„е®ҡд№үдҝЎжҒҜпјҢе№¶иҰҶзӣ–зҲ¶дҝЎжҒҜзҡ„дёҖдәӣй…ҚзҪ®еҖјгҖӮ

BeanDefinitionжҺҘеҸЈзҡ„ж–№жі•е®ҡд№ү

BeanDefinition ж•ҙдҪ“еҢ…еҗ«д»ҘдёӢеҮ дёӘйғЁеҲҶпјҡ

  • Bean зҡ„зұ»дҝЎжҒҜ - е…Ёйҷҗе®ҡзұ»еҗҚ ( beanClassName )
  • Bean зҡ„еұһжҖ§ - дҪңз”Ёеҹҹ ( scope ) гҖҒжҳҜеҗҰй»ҳи®Ө Bean ( primary ) гҖҒжҸҸиҝ°дҝЎжҒҜ ( description ) зӯү
  • Bean зҡ„иЎҢдёәзү№еҫҒ - жҳҜеҗҰе»¶иҝҹеҠ иҪҪ ( lazy ) гҖҒжҳҜеҗҰиҮӘеҠЁжіЁе…Ҙ ( autowireCandidate ) гҖҒеҲқе§ӢеҢ– / й”ҖжҜҒж–№жі• ( initMethod / destroyMethod ) зӯү
  • Bean дёҺе…¶д»– Bean зҡ„е…ізі» - зҲ¶ Bean еҗҚ ( parentName ) гҖҒдҫқиө–зҡ„ Bean ( dependsOn ) зӯү
  • Bean зҡ„й…ҚзҪ®еұһжҖ§ - жһ„йҖ еҷЁеҸӮж•° ( constructorArgumentValues ) гҖҒеұһжҖ§еҸҳйҮҸеҖј ( propertyValues ) зӯү

з”ұжӯӨеҸҜи§ҒпјҢBeanDefinition еҮ д№ҺжҠҠ bean зҡ„жүҖжңүдҝЎжҒҜйғҪиғҪж”¶йӣҶе№¶е°ҒиЈ…иө·жқҘпјҢеҸҜд»ҘиҜҙжҳҜеҫҲе…ЁйқўдәҶгҖӮ

:star::star::star::star::star:йқўиҜ•йўҳпјҡжҰӮиҝ°BeanDefinition

BeanDefinition жҸҸиҝ°дәҶ SpringFramework дёӯ bean зҡ„е…ғдҝЎжҒҜпјҢе®ғеҢ…еҗ« bean зҡ„зұ»дҝЎжҒҜгҖҒеұһжҖ§гҖҒиЎҢдёәгҖҒдҫқиө–е…ізі»гҖҒй…ҚзҪ®дҝЎжҒҜзӯүгҖӮBeanDefinition е…·жңүеұӮж¬ЎжҖ§пјҢе№¶дё”еҸҜд»ҘеңЁ IOC е®№еҷЁеҲқе§ӢеҢ–йҳ¶ж®өиў« BeanDefinitionRegistryPostProcessor жһ„йҖ е’ҢжіЁеҶҢпјҢиў« BeanFactoryPostProcessor жӢҰжҲӘдҝ®ж”№зӯүгҖӮ

BeanDefinitionзҡ„з»“жһ„

AttributeAccessor

еұһжҖ§зҡ„и®ҝй—®еҷЁпјҡе®ҡд№үз”ЁдәҺе°Ҷе…ғж•°жҚ®йҷ„еҠ еҲ°д»»ж„ҸеҜ№иұЎпјҢжҲ–д»Һд»»ж„ҸеҜ№иұЎи®ҝй—®е…ғж•°жҚ®зҡ„йҖҡз”ЁеҚҸе®ҡзҡ„жҺҘеҸЈгҖӮ

жҖ»з»“еҮәз¬¬дёҖдёӘ BeanDefinition зҡ„зү№еҫҒпјҡ**BeanDefinition з»§жүҝдәҶ AttributeAccessor жҺҘеҸЈпјҢе…·жңүй…ҚзҪ® bean еұһжҖ§зҡ„еҠҹиғҪгҖӮ**

BeanMetadataElement

еӯҳж”ҫдәҶ bean зҡ„е…ғдҝЎжҒҜпјҢ еҸӘжңүдёҖдёӘж–№жі•пјҢжҳҜиҺ·еҸ– bean зҡ„иө„жәҗжқҘжәҗ

иө„жәҗжқҘжәҗпјҢиҜҙзҷҪдәҶпјҢе°ұжҳҜ bean зҡ„ж–Үд»¶ / url и·Ҝеҫ„гҖӮе’ұд»¬еүҚйқўеҶҷзҡ„жүҖжңүзӨәдҫӢпјҢйғҪжҳҜеңЁжң¬ең°зЈҒзӣҳдёҠзҡ„ .class ж–Үд»¶еҠ иҪҪиҝӣжқҘзҡ„пјҢжүҖд»ҘеҜ№еә”зҡ„д№ҹе°ұеә”иҜҘжҳҜ FileSystemResource

AbstractBeanDefinition

еҲ°дәҶ BeanDefinition зҡ„з¬¬дёҖдёӘе®һзҺ°зұ»дәҶпјҢдҪңдёә BeanDefinition зҡ„жҠҪиұЎе®һзҺ°пјҢе®ғйҮҢйқўе·Із»Ҹе®ҡд№үеҘҪдәҶдёҖдәӣеұһжҖ§е’ҢеҠҹиғҪпјҲеӨ§йғЁеҲҶйғҪжңүдәҶпјүпјҢеӨ§дҪ“еҢ…еҗ«д»ҘдёӢеҶ…е®№пјҡпјҲеҸӘжҢ‘йҖүйғЁеҲҶйҮҚиҰҒеұһжҖ§пјүпјҡ

GenericBeanDefinition

еҸҲзңӢеҲ° Generic дәҶпјҢе®ғд»ЈиЎЁзқҖйҖҡз”ЁгҖҒдёҖиҲ¬зҡ„пјҢжүҖд»Ҙиҝҷз§Қ BeanDefinition д№ҹе…·жңүдёҖиҲ¬жҖ§гҖӮGenericBeanDefinition зҡ„жәҗз Ғе®һзҺ°йқһеёёз®ҖеҚ•пјҢд»…д»…жҳҜжҜ” AbstractBeanDefinition еӨҡдәҶдёҖдёӘ parentName еұһжҖ§иҖҢе·ІгҖӮ

з”ұиҝҷдёӘи®ҫи®ЎпјҢеҸҜд»Ҙеҫ—еҮәд»ҘдёӢеҮ дёӘз»“и®әпјҡ

  • AbstractBeanDefinition е·Із»Ҹе®Ңе…ЁеҸҜд»Ҙжһ„жҲҗ BeanDefinition зҡ„е®һзҺ°дәҶ
  • GenericBeanDefinition е°ұжҳҜ AbstractBeanDefinition зҡ„йқһжҠҪиұЎжү©еұ•иҖҢе·І
  • GenericBeanDefinition е…·жңүеұӮж¬ЎжҖ§пјҲеҸҜд»ҺзҲ¶ BeanDefinition еӨ„з»§жүҝдёҖдәӣеұһжҖ§дҝЎжҒҜпјү

RootBeanDefinitionдёҺChildBeanDefinition

root е’Ң child пјҢеҫҲжҳҺжҳҫиҝҷжҳҜзҲ¶еӯҗе…ізі»зҡ„ж„ҸжҖқдәҶе‘ҖгҖӮеҜ№дәҺ ChildBeanDefinition пјҢе®ғзҡ„и®ҫи®Ўе®һзҺ°дёҺ GenericBeanDefinition еҰӮеҮәдёҖиҫҷпјҢйғҪжҳҜйӣҶжҲҗдёҖдёӘ parentName жқҘдҪңдёәзҲ¶ BeanDefinition зҡ„ вҖңжҢҮеҗ‘еј•з”ЁвҖқ гҖӮдёҚиҝҮжңүдёҖзӮ№иҰҒжіЁж„ҸпјҢ ChildBeanDefinition жІЎжңүй»ҳи®Өзҡ„ж— еҸӮжһ„йҖ еҷЁпјҢеҝ…йЎ»иҰҒдј е…Ҙ parentName жүҚеҸҜд»ҘпјҢдҪҶ GenericBeanDefinition еҲҷжңүдёӨз§ҚдёҚеҗҢзҡ„жһ„йҖ еҷЁгҖӮ

AnnotatedBeanDefinition

е®ғеҸҜд»ҘжҠҠ Bean дёҠзҡ„жіЁи§ЈдҝЎжҒҜжҸҗдҫӣеҮәжқҘ

еҹәдәҺ@Componentзҡ„BeanDefinition

з»ҷ Person дёҠжү“дёҖдёӘ @Component жіЁи§ЈпјҢз„¶еҗҺдҪҝз”Ё AnnotationConfigApplicationContext жқҘй©ұеҠЁжү«жҸҸ Person зұ»

еҹәдәҺ xml и§ЈжһҗеҮәжқҘзҡ„ bean пјҢе®ҡд№үжқҘжәҗжҳҜ xml й…ҚзҪ®ж–Үд»¶пјӣеҹәдәҺ @Component жіЁи§Ји§ЈжһҗеҮәжқҘзҡ„ bean пјҢе®ҡд№үжқҘжәҗжҳҜзұ»зҡ„ .class ж–Үд»¶дёӯгҖӮ

еҹәдәҺ@Beanзҡ„BeanDefinition

зј–еҶҷдёҖдёӘй…ҚзҪ®зұ» BeanDefinitionQuickstartConfiguration пјҢдҪҝз”Ё @Bean жіЁеҶҢдёҖдёӘ Person пјҡ

д№ӢеҗҺпјҢдҪҝз”ЁиҝҷдёӘй…ҚзҪ®зұ»й©ұеҠЁ IOC е®№еҷЁпјҢе№¶зӣҙжҺҘиҺ·еҸ– BeanDefinition пјҡ

е…·дҪ“еҢәеҲ«еҸҜд»ҘеҸ‘зҺ°жңүиҝҷд№ҲеҮ дёӘпјҡ

  • Bean зҡ„зұ»еһӢжҳҜ Root bean пјҲ ConfigurationClassBeanDefinition з»§жүҝиҮӘ RootBeanDefinition пјү
  • Bean зҡ„ className дёҚи§ҒдәҶ
  • иҮӘеҠЁжіЁе…ҘжЁЎејҸдёә AUTOWIRE_CONSTRUCTOR пјҲжһ„йҖ еҷЁиҮӘеҠЁжіЁе…Ҙпјү
  • жңү factoryBean дәҶпјҡperson з”ұ beanDefinitionQuickstartConfiguration зҡ„ person ж–№жі•еҲӣе»ә

:star::star::star:гҖҗеҺҹзҗҶгҖ‘BeanDefinitionжҳҜеҰӮдҪ•з”ҹжҲҗзҡ„пјҲз®Җжҳ“зҗҶи§Јпјү

  1. йҖҡиҝҮ xml еҠ иҪҪзҡ„ BeanDefinition пјҢе®ғзҡ„иҜ»еҸ–е·Ҙе…·жҳҜ XmlBeanDefinitionReader пјҢе®ғдјҡи§Јжһҗ xml й…ҚзҪ®ж–Үд»¶пјҢжңҖз»ҲжқҘеҲ° DefaultBeanDefinitionDocumentReader зҡ„ doRegisterBeanDefinitions ж–№жі•пјҢж №жҚ® xml й…ҚзҪ®ж–Үд»¶дёӯзҡ„ bean е®ҡд№үжһ„йҖ  BeanDefinition пјҢжңҖеә•еұӮеҲӣе»ә BeanDefinition зҡ„дҪҚзҪ®еңЁ org.springframework.beans.factory.support.BeanDefinitionReaderUtils#createBeanDefinition гҖӮ
  2. йҖҡиҝҮжЁЎејҸжіЁи§Ј + з»„д»¶жү«жҸҸзҡ„ж–№ејҸжһ„йҖ зҡ„ BeanDefinition пјҢе®ғзҡ„жү«жҸҸе·Ҙе…·жҳҜ ClassPathBeanDefinitionScanner пјҢе®ғдјҡжү«жҸҸжҢҮе®ҡеҢ…и·Ҝеҫ„дёӢеҢ…еҗ«зү№е®ҡжЁЎејҸжіЁи§Јзҡ„зұ»пјҢж ёеҝғе·ҘдҪңж–№жі•жҳҜ doScan ж–№жі•пјҢе®ғдјҡи°ғз”ЁеҲ°зҲ¶зұ» ClassPathScanningCandidateComponentProvider зҡ„ findCandidateComponents ж–№жі•пјҢеҲӣе»ә ScannedGenericBeanDefinition е№¶иҝ”еӣһгҖӮ
  3. йҖҡиҝҮй…ҚзҪ®зұ» + @Bean жіЁи§Јзҡ„ж–№ејҸжһ„йҖ зҡ„ BeanDefinition жңҖеӨҚжқӮпјҢе®ғж¶үеҸҠеҲ°й…ҚзҪ®зұ»зҡ„и§ЈжһҗгҖӮй…ҚзҪ®зұ»зҡ„и§ЈжһҗиҰҒиҝҪиёӘеҲ° ConfigurationClassPostProcessor зҡ„ processConfigBeanDefinitions ж–№жі•пјҢе®ғдјҡеӨ„зҗҶй…ҚзҪ®зұ»пјҢе№¶дәӨз»ҷ ConfigurationClassParser жқҘи§Јжһҗй…ҚзҪ®зұ»пјҢеҸ–еҮәжүҖжңүж ҮжіЁдәҶ @Bean зҡ„ж–№жі•гҖӮйҡҸеҗҺпјҢиҝҷдәӣж–№жі•еҸҲиў« ConfigurationClassBeanDefinitionReader и§ЈжһҗпјҢжңҖз»ҲеңЁеә•еұӮеҲӣе»ә ConfigurationClassBeanDefinition е№¶иҝ”еӣһгҖӮ
дҪҝз”ЁжүӢжңәи®ҝй—®иҝҷзҜҮж–Үз«
